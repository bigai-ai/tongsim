# Voxel Perception

The voxel perception utilities allow clients to sample sparse volumetric data
around a specified region, which is useful for reinforcement learning and
planning workloads.

## Key Functions

- `query_voxel`: Capture a voxel buffer around a transform with configurable
  resolution and extents.
