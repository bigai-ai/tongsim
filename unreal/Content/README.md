# Unreal Engine Content (External)

Unreal Engine assets are no longer stored in this repository.

To use the Unreal project, download the assets separately and extract them into
`unreal/Content/`.

Download source:
- https://huggingface.co/datasets/bigai/tongsim-unreal-content

To fetch into this repo:
`python scripts/fetch_unreal_content.py`

This directory is intentionally kept empty in Git (only this `README.md` is
tracked).
